业务触发事件清单
//grid事件
domi.viewinfo    :   查看数据时事件
domi.add.before  : 新增按钮之前事件
domi.add.after  : 新增按钮之后事件
domi.modify.before  : 修改按钮之前事件
domi.modify.after  : 修改按钮之后事件
domi.delete.before  : 删除按钮之前事件
domi.delete.after  : 删除按钮之后事件
//表单事件
domi.addsave.before  : 新增保存之前事件
domi.addsave.after  : 新增保存之后事件
domi.addsave.failue : 新增保存失败事件
domi.modifysave.before  : 修改保存之前事件
domi.modifysave.after  : 修改保存之后事件
domi.modifysave.failue : 修改保存失败事件
domi.deletesave.before  : 删除保存之前事件
domi.deletesave.after  : 删除保存之后事件
domi.deletesave.failue : 删除保存失败事件
